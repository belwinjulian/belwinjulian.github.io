# Mission: Databricks Certified Data Engineer Associate

## Why
Pass the Databricks Certified Data Engineer Associate exam by July 2, 2026, and build practical Databricks skills ahead of a real work project. The certificate is the milestone; the real goal is being able to do the job.

## Success looks like
- Pass the Databricks Certified Data Engineer Associate exam on or before July 2, 2026
- Able to explain the Lakehouse architecture, Delta Lake, and Medallion pattern to a colleague
- Able to write and read PySpark and Spark SQL transformations confidently
- Able to build incremental pipelines using Auto Loader and Delta Live Tables
- Able to set up production jobs and understand Unity Catalog governance

## Exam version
**May 6, 2026 refactor** — all teaching must align to this version.
5 weighted domains, 45 questions, 90 min, SQL-first (Python fallback), no aids.
| Domain | Weight |
|---|---|
| Databricks Intelligence Platform | 10% |
| Development & Ingestion | 30% |
| Data Processing & Transformations | 31% |
| Productionizing Data Pipelines | 18% |
| Data Governance & Quality | 11% |

## Constraints
- 12 days to exam date (June 20 → July 2, 2026)
- Beginner: limited Python, limited SQL, no Spark, no cloud, no prior Databricks
- Prefers active recall: teach → brain dump → gap fill
- Mastery matters more than speed — don't rush past confusion

## Out of scope
- Deep Spark internals (JVM, query planning, performance tuning)
- MLflow and ML-specific Databricks features
- Infrastructure administration (cluster sizing, IAM, cost optimization)
- Other cloud certifications (AWS, Azure, GCP)
