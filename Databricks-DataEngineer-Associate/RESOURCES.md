# Databricks Data Engineer Associate Resources

## Knowledge

- [Databricks Certified Data Engineer Associate — Official Exam Page](https://www.databricks.com/learn/certification/data-engineer-associate)
  The authoritative source for exam format, topic weights, and the May 2026 exam guide PDF. Always check here first for any changes.

- [Delta Lake Documentation](https://docs.delta.io/latest/index.html)
  Primary source for Delta Lake concepts: ACID transactions, time travel, schema enforcement, transaction log. Use for Lessons 2–3.

- [Apache Spark Documentation — PySpark](https://spark.apache.org/docs/latest/api/python/)
  Official PySpark API reference. Use for any DataFrame operation questions.

- [Databricks Documentation](https://docs.databricks.com)
  The canonical reference for all Databricks-specific features: Auto Loader, Delta Live Tables, Lakeflow Jobs, Unity Catalog, Databricks CLI, Bundles. Use for every domain.

- [Databricks Academy — Data Engineer Learning Path](https://www.databricks.com/learn/training/home)
  Free self-paced courses directly from Databricks. Aligned to the exam. Use to supplement each lesson with hands-on labs.

- [Certsafari — May 2026 Exam Domain Breakdown](https://www.certsafari.com/databricks/data-engineer-associate)
  Clean breakdown of all 7 exam domains and their subtopics from the May 2026 version. Use to verify lesson coverage.

## Gaps

- No high-quality free practice exam found yet aligned to the May 2026 refactor — third-party dumps (ExamTopics, SkillCertPro) may contain old-version questions. Treat with caution.

## Wisdom (Communities)

- [Databricks Community Forums](https://community.databricks.com)
  Official forum. Good for real-world questions about platform behavior and gotchas.

- [r/databricks](https://reddit.com/r/databricks)
  Active community with exam prep threads, tips, and real-world usage discussion.
